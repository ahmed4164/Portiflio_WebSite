# Ahmed's Portfolio

A modern React portfolio website showcasing mobile app development skills and projects.

## 🚀 Live Demo

[View Portfolio](https://ahmed011.netlify.app/)

## 🛠️ Tech Stack

- React 18
- Framer Motion
- React Icons
- CSS3

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```
